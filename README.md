Теорема Пифагора

Формула :
c² = a² + b² , отсюда следует , что  
c = √(a² + b²) 


Интерактивная 3D-визуализация теоремы Пифагора. Один прямоугольный треугольник, два катета (a, b) и гипотенуза (c) — можно менять любое значение,оно постоянно будет пересчитываться , и равенство a² + b² = c² всегда выполняется.
Сдела для того чтобы наглядно показать  своему брату как работает теорема Пифагора

3D модель можно двигать , вращать , менять длину катетов и значение всегда будет пересчитываться 

Потрогать проект можно тут  - https://skyproger.github.io/Pythagoras-3D/
